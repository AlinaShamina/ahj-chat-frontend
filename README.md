# AHJ Chat (WebSockets)

![Build](https://github.com/AlinaShamina/ahj-chat-frontend/actions/workflows/deploy.yml/badge.svg)

Простой чат на WebSockets.  
Позволяет отправлять сообщения в реальном времени и видеть активных пользователей.

## Ссылки

- **Frontend:** https://alinashamina.github.io/ahj-chat-frontend  
- **Backend (Render):** https://ahj-chat-backend-y89x.onrender.com

## Использование

1. Открой фронтенд по ссылке.  
2. Введи уникальный никнейм.  
3. Общайся с другими пользователями в чате.  
4. Сообщения отправляются по Enter, свои — справа, чужие — слева.
